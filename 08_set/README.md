## set이 제공하는 함수 종류
|함수명|	설명|
|------|------|
|intersection(x)| 현재 set과 파라미터 x set과의 교집합|
|union(x)|현재 set과 파라미터 x set과의 합집합|
|difference(x)|현재 set과 파라미터 x set과의 차집합|
|add(x)	|값 x를 추가|
|update(리스트)	|값 여러 개를 추가|
|remove(x)	|x 요소 제거|
|clear() | 모든 요소 제거|
