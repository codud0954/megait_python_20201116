## 파일 읽기, 쓰기
`217p`

## 파일 입출력
```
open(file_path, mode, encoding)
```
* file_path: 파일 경로
* mode 
  * w: 쓰기
  * a: 기존 파일에 추가하기
  * r: 읽기
* encoding: 문자 인코딩(euc-kr, utf8 등)
